Package me.axolotldev.dbconnector.abstracts.database

Interface Connector

  • All Implemented Interfaces:
    java.lang.AutoCloseable 
    
public interface Connector
 implements AutoCloseable

    Represents a basic database connector interface for executing SQL operations.

    Implementations are responsible for managing the database connection lifecycle and executing SQL queries via statement objects.
